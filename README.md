# Kalendar für die Sennesperrzeiten, direkt im Handy
Für Schlangen, Oesterholz-Haustenbeck, Bad Lippspringe, Sennelager, Augustdorf und Detmold

## Kalender-Abonnement zum iPhone hinzufügen

1. Öffnen Sie die Kalender-App auf Ihrem iPhone.

2. Tippen Sie unten auf dem Bildschirm auf "Kalender" oder das Kalender-Symbol

3. Tippen Sie unten links auf "Kalender hinzufügen".

4. Tippen Sie auf "Kalender-Abo hinzufügen".

5. Fügen Sie die URL https://raw.githubusercontent.com/andreashochhalter/senne-kalender/refs/heads/main/senne_calendar.ics in das Feld "Server" oder "URL" ein.

6. Tippen Sie auf "Suchen" oder "Abonnieren".

7. Möglicherweise werden Sie aufgefordert, den Kalender zu benennen und Einstellungen vorzunehmen. Tippen Sie zum Abschluss auf "Hinzufügen" oder "Fertig".

------
## Add Calendar Subscription to iPhone

1. Open the Calendar app on your iPhone.

2. Tap the Calendars button at the bottom of the screen.

3. Tap Add Calendar in the bottom-left corner.

4. Tap Add Subscription Calendar.

5. Paste the URL https://raw.githubusercontent.com/andreashochhalter/senne-kalender/refs/heads/main/senne_calendar.ics into the "Server" or "URL" field.

6. Tap Find or Subscribe.

7. You may be prompted to name the calendar and choose its settings. Tap Add or Done when finished.
